# Obsidian Open Terminal

Minimal Obsidian desktop plugin that adds an **Open in terminal** item to the file explorer context menu.

GitHub: [spicyneuron/obsidian-open-terminal](https://github.com/spicyneuron/obsidian-open-terminal)

## What it does

- Right-click a file to open its parent folder in your terminal.
- Right-click a folder to open that folder in your terminal.
- Configure the terminal app or command for the current platform in plugin settings.

## Development

- `pnpm install`
- `pnpm dev`
- `pnpm build`
